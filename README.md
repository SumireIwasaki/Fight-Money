## 1.アプリ概要

Fight Moneyは、親子でお手伝いとおこづかいを管理するWebアプリです。

お手伝いの登録から完了報告、保護者による承認、おこづかいの管理までを一つのアプリで行うことができます。

子どもの頑張りを見える化し、お手伝いを楽しく継続できる環境を目指して制作しました。

## 2.開発背景

小さい頃、お手伝いを通して継続する力や頑張る力を身につけた経験がありました。

しかし、現在は共働き家庭の増加などにより、親子で一緒にお手伝いをする機会が減っています。

そこで、

「頑張る力を、当たり前に。」

をコンセプトに、お手伝いをもっと楽しく続けられるアプリとしてFight Moneyを制作しました。

## 5.主な機能

-👨‍👩‍👧 親子アカウント：
  親・子どものログイン、
  QRコードで家族登録
-📅 お手伝い管理：
  カレンダー表示、
  お手伝い追加、
  当日の予定表示
-✅ 完了報告：
  子どもがお手伝い完了、
  保護者へ通知
-💰 おこづかい管理：
  保護者が承認、
  自動でおこづかいに反映、
  月・年ごとの集計

## 7.制作を通して学んだこと

Fight Moneyの制作では、HTML・CSS・JavaScriptだけでなく、FlaskやSQLiteを用いたWebアプリ開発を経験しました。

今回の制作では、企画から設計、デザイン、実装、公開までを一貫して行ったことで、アプリ開発全体の流れを学ぶことができました。ユーザーの課題を考え、それをどのような機能で解決するかを検討しながら制作を進めたことで、技術だけでなく企画力や設計力の大切さも実感しました。

また、今回初めてデータベース（SQLite）を利用し、親と子どもという複数のユーザー間で情報を共有する仕組みの実装に挑戦しました。 お手伝いの登録、完了報告、保護者による承認、おこづかいへの反映までの一連の流れをデータベースで管理することで、ユーザー同士が同じ情報を共有できる仕組みを構築しました。この経験を通して、データベースの役割や、Webアプリにおけるデータ管理の重要性を学ぶことができました。

さらに、Git・GitHubを用いたバージョン管理や、GitHub Pagesを利用したWeb公開にも取り組みました。
